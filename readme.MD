Synchronize - веб-приложение с открытым исходным кодом для людей, которые хотят поделиться файлами в защищённом виде.
# Что делает Synchronize?
Synchronize шифрует файл (файлы) ассиметричным алгоритмом RSA, после чего выдаёт пользователю 2 возможности: 
+ Скачать зашифрованный файл
	+ В этом случае пользователь скачает зашифрованный файл `.synchronize`, который можно будет передавать куда угодно, не боясь за получение к содержимому. Расшифровать сможет только человек, имеющий ключ
+ Поделиться файлом
	+ В этом случае софт автоматически выгрузит файл на доступные облачные хранилища и файлообменники
		Пользователь должен быть авторизован в них, ведь сохраняться всё будет на его аккаунт.
Разумеется, расшифровать файлы в Synchronize тоже можно
# Аналоги
